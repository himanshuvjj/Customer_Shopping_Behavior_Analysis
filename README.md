# 📊 Customer Shopping Behavior Analysis Dashboard

## Overview

This project focuses on analyzing customer behavior data using Python, SQL, and Power BI to uncover actionable business insights. The workflow covers the complete analytics lifecycle—from data collection and cleaning to interactive dashboard development and stakeholder reporting.

The objective is to identify key trends, customer patterns, and performance metrics that support data-driven decision-making.

---

## 🎯 Project Objectives

- Load and process raw customer data using Python
- Perform Exploratory Data Analysis (EDA)
- Clean and transform data for analysis
- Execute analytical SQL queries
- Build an interactive Power BI dashboard
- Generate insights and recommendations
- Create a presentation summarizing findings using Gamma

---

## 🗂️ Dataset

**Dataset Name:** Customer Shopping Behavior Dataset

### Key Features

- Customer ID
- Demographics
- Purchase History
- Product Categories
- Transaction Amount
- Order Frequency
- Location Information
- Customer Segments

> **Note:** The dataset used in this project is intended for educational and analytical purposes only.

## 🔄 Project Workflow

### 1. Data Loading

- Imported dataset using Python
- Validated data structure and data types
- Checked dataset dimensions and column information

### 2. Exploratory Data Analysis (EDA)

- Generated summary statistics
- Analyzed distributions and trends
- Identified correlations and patterns
- Created visualizations for key metrics

### 3. Data Cleaning

- Handled missing values
- Removed duplicate records
- Standardized column names
- Corrected data types
- Treated outliers where necessary

### 4. SQL Analysis

Executed SQL queries to answer business questions such as:

- Top-performing customer segments
- Revenue by category
- Customer retention patterns
- Monthly sales trends
- High-value customers

### 5. Dashboard Development

Built an interactive Power BI dashboard featuring:

- KPI cards
- Trend analysis
- Customer segmentation
- Geographic insights
- Filter and drill-down functionality

### 6. Reporting

- Summarized findings in a business report
- Highlighted recommendations and next steps
- Created a presentation using Gamma
